# Controls

- Joysticks
    - <kbd>Left Joystick</kbd> - Drive
    - <kbd>Right Joystick</kbd> - Rotate
    - <kbd>Left Joystick (Press in)</kbd> - Swap Coral/Algae
    - <kbd>Right Joystick (Press in)</kbd> - Coral Station Collect
- Bumpers & Triggers
    - <kbd>Left Bumper</kbd> - Toggle Collector Deployment
    - <kbd>Right Bumper</kbd> - Start Auto Score
    - <kbd>Left Trigger</kbd> - Spin Intake
    - <kbd>Right Trigger</kbd> - Place Coral
- ABXY
    - <kbd>A</kbd> - Select L2
    - <kbd>B</kbd> - Select L3 / Processor Algae Mode
    - <kbd>X</kbd> - Override Left-Right Reef Placement (?)
    - <kbd>Y</kbd> - Go to L4 / Net Algae Mode
- DPad
    - <kbd>DPad Left</kbd> - Eject
    - <kbd>DPad Right</kbd> - Field/Robot Orientation
    - <kbd>DPad Down</kbd> - Reset Heading
- Other
    - <kbd>Start</kbd> - Climb (2-press)
    - <kbd>Back</kbd> - Climb Cancel
